# 概要

ジョブカンのUIをあまりにも美しくします。

# How to Use

1. chrome://extensions/ を開く
2. デベロッパーモードを有効にする
3. パッケージ化されていない拡張機能を読み込む
4. manifest.jsonのあるフォルダを選択
5. jobcan勤怠の月単位編集のページを開く
6. 美しさに震える
